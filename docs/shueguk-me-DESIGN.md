# 슈국 학생 라운지 (`shueguk-me`) — 설계 문서

> 이수경국어(Shueguk) **학생 전용 앱** 구상 정리.
> 학생이 로그인하면 자기 성적표·공지·활동·계급을 **슈국 디자인**으로 보는 PWA.
>
> _이 문서는 기획 단계의 지도입니다. (작성: 설계 논의 정리본)_

---

## 1. 한 줄 목표

> 🎯 학생이 **「이름 + 학부모님 전화 가운데 4자리」** 로 로그인하면,
> 자기 **개인 페이지**(성적표·공지·활동·계급)에 바로 들어가고,
> 그것을 **노션 느낌이 아니라 슈국 스타일**로 예쁘게 보여준다.

현재는 선생님이 학생마다 노션 링크를 손으로 공유 중 → 이를 **로그인 한 번으로 자동화 + 슈국 디자인**으로 발전.

---

## 2. 현재 상황 (출발점)

### 기존 자산
- **shueguk-hub** — 👩‍🏫 선생님용 허브(PWA). 어휘/문법 테스트, H WORK(숙제), 주말 실전모의고사, 클리닉, 지필고사 리포트 메뉴.
- **shueguk-clinic** — 🧑‍🎓 학생용 클리닉 신청 폼. → **Google Apps Script** 웹앱으로 제출(JSONP) → 구글 시트 적재.
- **shueguk-report** — 지필고사 리포트 도구. *(아직 본격 사용 전)*
- 공통: 전부 GitHub Pages + 정적 HTML(PWA), Pretendard/Gowun 폰트, 초록 팔레트.

### 노션 구조 (스크린샷 기준)
```
📄 학생 페이지 "박하율"   (학생 1명 = 노션 페이지 1개)
   ├─ 속성: 학교=능곡고, 학년=2026 고2   ← ⚠️ 전화번호 칸 없음
   ├─ 🗓️ 정규 기간반 / 내신 기간반   (가/나/진도/확인 수업 + 시간)
   ├─ 📢 공지사항 "Shueguk에서 알려드립니다"  (공개 — 모두 공유)
   ├─ 💡 개별 안내   (이 학생만, 현재 비어있음)
   └─ 💯 성적 데이터   ← 시험 목록
        └─ 26-1-중간-지필고사 (날짜, 분류=내신)
             └─ 📎 [분석지] 스캔 PDF  ← 점수·분석·손편지가 여기 (이미지)
```

### 핵심 제약 ⚠️
- 성적표는 **한글(HWP)로 만들어 출력 → 손편지 추가 → 스캔한 이미지 PDF**.
- 즉 **점수가 숫자 데이터로 어디에도 저장돼 있지 않음** (PDF 그림 안에만 존재).
- → **자동 성적 그래프·등수는 현재 불가능.** 점수를 숫자로 입력하기 시작하면 그때 가능.

## 3. 전체 아키텍처

```
        ┌─────────── 구글 시트들 ──────────┐
        │ 어휘 · 문법 · H WORK · 모의고사 · 클리닉 │   ← 활동/참여 데이터 (허브 도구가 이미 적재)
        │ + 학생 명단(엑셀 업로드)              │   ← 마스터 로스터
        └──────────────┬───────────────────┘
                       │
   Notion ────────────┤   ← 성적표 PDF · 공지 · 개별안내 · 수업시간
                       │
                       ▼
              ┌─────────────────┐
              │  하나의 브릿지     │  Google Apps Script
              │ (시트 + Notion)   │  · 비밀 토큰 숨김
              │                  │  · "이 학생 것만" 응답
              │                  │  · (후속) 포인트 계산 → 계급 판정
              └────────┬────────┘
                       │ JSONP / fetch
                       ▼
              ┌─────────────────┐
              │  학생 라운지 (PWA) │  GitHub Pages, 슈국 디자인
              └─────────────────┘
```

**왜 이 구조인가**
- 노션 토큰을 웹페이지에 직접 두면 전체 성적이 노출됨 → **토큰을 가진 브릿지가 가운데서 "요청한 학생 한 명"만 잘라서** 응답.
- Apps Script는 구글 시트 옆에 살면서 **시트(활동 데이터)와 Notion(성적표) 둘 다** 접근 가능 → 단일 브릿지로 통합.
- 노션 = 보이지 않는 콘텐츠 창고(헤드리스 CMS), 앱 = 슈국 디자인 매장. **선생님 작업 습관(노션에 PDF 업로드)은 그대로 유지.**

### 참고 (기술 메모)
- 노션 API의 업로드 파일 URL은 **임시(약 1시간 만료)** → 브릿지가 학생이 열 때마다 **새 URL을 발급받아** 전달.
- 노션 인티그레이션을 **학생 페이지들이 있는 영역에 공유**해야 API가 읽을 수 있음.

## 4. 주춧돌: 학생 명단(마스터 로스터)

포인트 합산과 로그인 모두 *"이 모든 기록이 같은 학생"* 임을 알아야 함 → **단일 명단 + 안정적 ID**가 핵심.

- 출처: **엑셀 업로드** → 구글 시트 한 장으로 보관(브릿지 옆, 갱신 쉬움).

| 칸 | 쓰임새 | 예시 |
|---|---|---|
| 이름 | 로그인 + 표시 | 박하율 |
| 학교 | 표시·구분 | 능곡고 |
| 학년 | 표시·구분 | 2026 고2 |
| **학부모 전화 가운데 4자리** | 🔑 로그인 열쇠 | 1234 |
| **노션 개인페이지 링크/ID** | 그 학생 성적표를 불러오는 주소 | notion.so/… |
| (선택) 학생코드 | 포인트 합산용 고유 ID | SG-0001 |

> 두 칸이 핵심: **전화4자리 = 문 여는 열쇠**, **노션 링크 = 학생 방 주소**.
> 학생코드는 지금 비워둬도 됨 → 3단계(포인트)에서 활용.

---

## 5. 로그인 & 보안

- 방식: **이름 + 학부모님 전화 가운데 4자리** (학생이 쓰기 가장 쉬움).
- 흐름: 입력 → 브릿지가 명단에서 매칭 → 해당 노션 페이지 콘텐츠 반환.
- 보안 (미성년자 성적 = 민감):
  - 브릿지에 **요청 횟수 제한**(예: 같은 기기 5회 오류 시 잠시 잠금) — 추측 공격 방지.
  - 한 번 로그인하면 **기기에 토큰 저장 → 다음부턴 자동 입장** (평소 자물쇠 거의 안 만남).
  - 응답은 **항상 본인 데이터만** — 전체 명단/타인 성적을 클라이언트로 보내지 않음.

## 6. 화면 구성

```
🔐 로그인  (이름 + 전화 가운데 4자리)
        │
        ▼
🏠 내 페이지  (전부 슈국 디자인)
   ├─ 📢 공지사항        (공개 DB 그대로)
   ├─ 💌 내 성적표        ← 스캔 PDF 목록 → 탭하면 폰에서 바로 열람
   ├─ 💡 개별 안내        (나에게만 온 메시지)
   ├─ 🗓️ 내 수업 시간     (정규/내신 기간반)
   ├─ 🧩 내 활동          ← 어휘·문법·H WORK·모의고사·클리닉 결과 (2단계)
   └─ 🏅 내 계급/포인트   ← ㄱㄴㄷ 계급 배지 (3단계)
```

디자인: 기존 클리닉/허브 톤 유지 — Pretendard / Gowun Batang, 초록 팔레트(`#3A6B5E` 계열), 둥근 카드, PWA 설치 지원.

## 7. 로드맵

| 단계 | 내용 | 새로 필요한 것 |
|---|---|---|
| **1. 입장** | 로그인 → 내 성적표 PDF + 공지 보기 | 학생 명단(엑셀) + 전화번호 칸 + 노션 링크 칸 |
| **2. 집결** | 내 어휘·문법·H WORK·모의고사·클리닉 결과 모아 보기 | 각 도구가 **같은 이름표(학생ID)** 로 기록하도록 점검 |
| **3. 게임화** | 참여 → 포인트 → **ㄱㄴㄷ 계급** 배지 | 포인트 규칙 + 계급 사다리 정의 |

### 1단계 확정 흐름 ✅
```
학생: 이름 + 전화4 입력
  → 브릿지: 시트 명단에서 본인 찾기
  → 브릿지: 그 노션 페이지에서 성적표 PDF + 공지 가져오기
  → 앱: 슈국 디자인으로 보여주기
```

---

## 8. 게임화: 포인트 & 계급 (3단계)

- 참여 활동을 포인트로 환산 (예: H WORK 제출 +10P, 클리닉 참여 +20P, 어휘/문법 응시 +α).
- 누적 포인트로 **국어 자모 계급 사다리** 상승:

> 🥚 **기역반**(입문) → **니은반** → **디귿반** → … → 👑 **히읗반**(최고)
>
> 예시 표시: *"현재 당신의 레벨은 〈니은 학습반〉입니다."*

- 전제: 2단계의 통합 학생ID가 있어야 활동을 정확히 합산 가능.
